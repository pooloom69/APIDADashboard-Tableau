# APIDA Student Demographics Dashboard

## Overview
This Tableau dashboard visualizes APIDA (Asian Pacific Islander Desi American) student enrollment and demographics at Fullerton College.  
It was developed during my internship with the Office of Institutional Effectiveness to support equity-focused, data-informed decision making.

## Features
- Interactive filters by year, gender, and student subgroup  
- Trend analysis of enrollment over multiple terms  
- Visual breakdown of APIDA subgroup representation  
- Shareable public dashboard for administrators and faculty  

## Tech Stack
- Tableau Public  
- Excel / SQL for data preparation
